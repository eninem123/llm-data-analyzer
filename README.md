# LLM 数据分析工具（RAG 原型）

> **35岁前数仓工程师 → AI 数据工程师转型实战项目**

## 项目功能
- 调用 Kimi API 分析订单数据
- 自动生成用户消费柱状图
- 智能修改简历（AI 导向）
- RAG 聊天机器人：无限问答（简历 + 数据）

## 技术栈
- Python + Pandas + Matplotlib
- Kimi API（免费 200万 token）
- RAG 基础实现（知识库检索）
